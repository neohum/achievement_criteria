# 성취기준 분류 서비스 개발 완료 보고서 (Walkthrough)

요청하신 "초등학교 3~6학년군 성취기준 보드" 웹 애플리케이션의 1차 개발이 성공적으로 완료되었습니다. 기획된 모든 목표 기능을 충족하며 `Next.js` 프로덕션 빌드까지 검증되었습니다.

## 🛠 사용된 주요 기술 스택
- **프레임워크**: Next.js 14+ (App Router, Server API)
- **언어 및 스타일**: TypeScript, Tailwind CSS
- **데이터 저장 / DB**: PostgreSQL + Prisma ORM
- **실시간 오토세이브**: Redis (Upstash)
- **드래그 앤 드롭**: `@hello-pangea/dnd`
- **데이터 파싱 / 내보내기**: `papaparse` (CSV), `xlsx` (Excel)

## ✨ 구현된 핵심 기능
1. **다중 작업 공간 (로비 및 멀티 보드)**
   - 나만의 여러 "보드(작업 공간)"를 생성하고 서로 전환하며 독립적으로 성취기준 보드를 관리할 수 있습니다.
2. **구글 스프레드시트 데이터 연동**
   - 접속 시 실시간으로 지정된 구글 스프레드시트 CSV 데이터를 파싱합니다.
   - 우측 사이드바에서 `학년군`과 `과목` 조건에 따라 카드를 동적으로 필터링하여 보여줍니다.
3. **드래그 앤 드롭 (카드 배치)**
   - 오른쪽 패널에 있는 기준들을 중앙 "내 성취기준 보드"로 끌어와 자유롭게 배치할 수 있습니다.
   - 배치된 카드 간의 순서를 마우스로 쉽게 위/아래로 변경할 수 있습니다.
   - 메인 보드 하단 "삭제 영역"으로 드래그 하거나 카드별 [삭제] 버튼을 눌러 목록에서 제외할 수 있습니다.
4. **강력한 실시간 다중 협업 통신 (SSE & Redis Pub/Sub)**
   - 유료 외부 서비스를 배제하고, `Server-Sent Events(SSE)` 기술과 `Redis`의 메시지 브로커(Pub/Sub) 기능을 활용하여 **초저지연 실시간 협업 통신** 파이프라인을 독자적으로 구축했습니다.
   - 동일한 URL로 **여러 명이 동시 접속 시**, 상단에 실시간 `동시 접속자(N명 시청 중)`가 시각 표시됩니다.
   - 누군가 카드를 옮기거나 텍스트를 고치면 서버를 통해 다른 참여자 브라우저에 **0.1초 즉시 동기화**됩니다.
5. **각 카드별 메모 (수업 적용 계획 등)**
   - 메인 보드에 안착된 각 카드에는 메모를 작성할 수 있는 텍스트 입력창이 지원되며, 입력 즉시 자동 저장 및 공유됩니다.
6. **엑셀 다운로드 (Export)**
   - 작업 완료 후 우측 상단의 `[엑셀 다운로드]` 버튼을 클릭하면, 내가 정렬한 순서, 성취기준 상세, 그리고 기록한 메모들이 모두 들어있는 `.xlsx` 엑셀 파일로 1초 만에 만들어집니다.

## ✅ 추가된 프로젝트 환경 설정
- 소스코드와 생성된 모든 문서 리포트들은 요청하신 대로 `docs` 폴더(`docs/implementation_plan.md`, `docs/task.md`)에 잘 적재해 두었습니다.
- 기밀 정보(데이터베이스 및 레디스 URL)가 담겨있는 `.env` 파일은 깃(Git) 업로드 시 유출되지 않도록 `.gitignore`에 등록 처리되었습니다.

## 🚀 실행 및 테스트 방법
개발 환경에서 애플리케이션을 구동하여 직접 테스트하실 수 있습니다. (터미널에서 아래 명령어 입력)

```bash
npm run dev
```

터미널 하단에 `http://localhost:3000` 주소가 나타나면 `[Cmd]` 키를 누른 채 클릭하여 브라우저에서 실행해 주세요. 화면상에서 **우측 패널의 필터를 선택**하고 카드를 끌어당겨 보세요!
